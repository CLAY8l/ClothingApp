# ClothingApp

> **Android App for selling men's clothing**

## Purpose
ClothingApp is designed to help sell men's clothing online, exclusively for Android devices. The app provides an easy-to-use interface for browsing, selecting, and purchasing men's fashion items.

## Features
- Browse a curated selection of men's clothing
- View product details, images, and prices
- Add items to cart and proceed to checkout
- User authentication and simple profile management (customize as features expand)

## Installation

### Prerequisites
- Android Studio IDE
- Git (optional; see exception below)

### Cloning the Repository

To clone this repository using Git, open your terminal or command prompt and run:
```bash
git clone https://github.com/CLAY8l/ClothingApp.git
```
This will create a local copy of the project in your current directory.

**Exception: If you don't have Git installed**

You can download the project as a ZIP file directly:
1. Go to [https://github.com/CLAY8l/ClothingApp](https://github.com/CLAY8l/ClothingApp).
2. Click the green `Code` button.
3. Select `Download ZIP`.
4. Extract the ZIP file on your computer.

### Steps to Install and Run in Android Studio

1. **Open Project in Android Studio**
   - Launch Android Studio.
   - Click on `Open an Existing Project`.
   - Select the cloned or extracted `ClothingApp` directory.

2. **Build and Run**
   - Allow Android Studio to sync the Gradle files.
   - Connect your Android device or start an emulator.
   - Click `Run` to build and launch the app.
